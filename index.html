General

Docs only

PsychoPy is an open-source application to allow the presentation of stimuli and collection of data for a wide range of neuroscience, psychology and psychophysics experiments. It’s a free, powerful alternative to Presentation™ or e-Prime™, written in Python (a free alternative to Matlab™ ).

News¶

Latest version: 1.83.04 was released Feb 2016. See the full Changelog

We’ve got a new Movie engine!! Rejoice! MovieStim3 (built on the moviepy package) seems more robust and doesn’t require avbin or vlc to be installed separately. In fact, if you don’t have the necessary libs it should go and fetch the right one for you!
